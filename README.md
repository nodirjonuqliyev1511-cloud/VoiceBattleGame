# Voice Battle

Игра для ведущего: участники орут в микрофон телефонов, ведущий видит гонку техники
(машина/джип/трактор/танк) в реальном времени — кто громче орёт, тот вырывается вперёд.

## Запуск

```
npm install
npm start
```

Откроется адрес вида `https://localhost:3000/host.html` — это экран ведущего.
QR-код на экране ведёт участников на `join.html` с уже подставленным кодом комнаты.

## Предупреждение браузера "соединение не защищено"

Сервер использует HTTPS с самоподписанным сертификатом — это обязательное требование
браузеров для доступа к микрофону (без HTTPS телефон не даст доступ к записи звука).
Из-за самоподписанного сертификата при первом заходе браузер один раз показывает
предупреждение — это нормально, нужно нажать "Дополнительно" → "Перейти на сайт".

### Как убрать это предупреждение совсем

Если хочется, чтобы гости заходили без единого лишнего клика — можно пробросить сервер
через бесплатный туннель Cloudflare, у которого уже настоящий доверенный сертификат:

1. Скачай `cloudflared` с официального сайта Cloudflare (раздел Cloudflare Tunnel,
   есть версия под Windows).
2. Запусти сервер как обычно (`npm start`), оставь окно открытым.
3. В новом окне терминала выполни:
   ```
   cloudflared tunnel --url https://localhost:3000 --no-tls-verify
   ```
4. Cloudflared выдаст ссылку вида `https://случайные-слова.trycloudflare.com`.
5. Открой экран ведущего по адресу вида
   `https://localhost:3000/host.html?tunnel=https://случайные-слова.trycloudflare.com`
   (вставь свою ссылку из шага 4 после `?tunnel=`) — QR-код автоматически перестроится
   на эту ссылку, и предупреждение у гостей исчезнет.

Важно: для этого способа нужен интернет на компьютере ведущего (сам туннель идёт через
интернет, даже если участники физически рядом). Если интернета нет или не хочется
возиться с туннелем — просто используй обычный QR, предупреждение нажимается один раз
за вечер на каждом телефоне.

## Настройки

В `server.js`:
- `BATTLE_DURATION_MS` — длительность раунда крика (сейчас 30000 = 30 сек)
- `COUNTDOWN_SECONDS` — отсчёт перед стартом (сейчас 3)
- `DISTANCE_KM_PER_SECOND_AT_MAX` — насколько быстро растёт "пройденная дистанция"
  в статистике победителя при максимальном крике
- `CAR_COLORS` / `VEHICLE_TYPES` — палитра цветов и набор типов техники,
  которые по кругу раздаются подключившимся игрокам
